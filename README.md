# Streambox

A Netflix clone video streaming application built for the INF web development course.

## Overview

Streambox is a modern video streaming platform that mimics Netflix's core functionality. Browse, search, and watch your favorite movies and TV shows in a sleek, responsive interface.

## Features

- **Browse Content** - Discover movies and TV shows with an intuitive interface
- **Search Functionality** - Easily find what you want to watch
- **Video Streaming** - Smooth playback of video content
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **User-Friendly Interface** - Clean, modern design inspired by Netflix

## Usage

1. Open the application in your browser
2. Browse available movies and TV shows
3. Use the search bar to find specific content
4. Click on a title to watch the video
